# GreenTech-MVP-Sostenible_Raul_Molina_Cordones

Se han renombrado las clases CSS .hero-gigante y .btn-eco-redundante a .hero y .btn-eco, respectivamente, para mejorar la claridad y evitar redundancias.
Se han eliminado las dependencias externas jQuery y Moment.js.
Se ha sustituido el uso de Moment.js por JavaScript nativo (Date) para obtener el año actual.
Se ha eliminado la inserción dinámica del footer mediante JavaScript.
Se ha añadido el footer directamente en el HTML, incluyendo un <span> con id year para actualizar el año.
Se ha simplificado el script, eliminando el uso de $(document).ready.
Se ha eliminado la carga innecesaria de múltiples pesos de la fuente, reduciéndolos a los estrictamente necesarios.
Se ha añadido una regla CSS específica para el footer en lugar de estilos inline.
Se ha eliminado código redundante y se ha mejorado la organización general del archivo.
